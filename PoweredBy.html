Where do I get this Engine?

The Engine is Open Source Under Github and NPM.
To install via NPM you do 'npm install [--global] hwe' or 'npm install [--global] --verbose hwe'. Choose Which one you need. If it's in a package you are working remove the global flag and run it. Otherwise run it with the global flag. Notice the [] means optional.
